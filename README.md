
# Water Level Alert System 💧

An ESP32-based IoT system for real-time water tank level monitoring with an Android application and intelligent water-level alerts.

## Features

* Real-time water level monitoring
* ESP32-based wireless communication
* Android app for live monitoring
* JSON-based data transmission
* Water level percentage and status display
* 90% threshold alert with repeated-alert prevention

## Technologies

**ESP32 | C++ | Wi-Fi | HTTP | JSON | Kotlin | Jetpack Compose | Android Studio**

## Working

The ESP32 measures the water level using an ultrasonic sensor and sends the processed data wirelessly to the Android application. The app displays live tank data and triggers an alert when the water level reaches 90%.
